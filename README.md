## Deployment

p1 & p2

```bash
  vagrant up
```

p3

```bash
  vagrant up
```

then you will need to connect with the virtual machine

```bash
  vagrant ssh
```
then run the scripts
```bash
  sh /scha/scripts/init.sh
```
```bash
  sh /scha/scripts/setup.sh
```
```bash
  sh /scha/scripts/deploy.sh
```
